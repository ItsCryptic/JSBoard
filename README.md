# JSBoard

JSBoard is a modern, fast, and community first forum software written using React & Next.
Currently JSBoard is in development and is **NOT** stable for production.
JSBoard is designed to be simple to use while also offering extensive configuration and management.

# Bugs

If you belive you have found a bug then do the following: - 1: Create a clean install of JSBoard - 2: Attempt to reproduce the bug - 3: If you can reproduce it then report the bug to us
When filling out a bug report follow the JSBoard issue templates instead of deleting them.
Make sure to use the correct type of template for the issue.
